Better Alternatives

Codenamed FUCK X, USE Y

There are many (high quality) alternatives to some proprietary programs you use everyday. Here is a small list of alternatives to some. Remember, these are just alternatives, not all are meant to be replacements, that is up for you to decide.


    FUCK ADOBE, USE

    The following reasons apply: Not Adobe

Krita

Why? While Krita itself it meant to be a digital painting program, it can still be useful in general photo editing similar to Photoshop. It's also a great alternative to Clip Studio Paint. It is completely free and open source, so there isn't any reason not to try it out. Supports opening .psd files.

Alternative to Photoshop

Photopea

Why? Photopea is browser based, and has a lot of Photoshop's functionality, all in your browser, all for free. It also offers some small but nice features, like a benchmark, cloud storage, and scripts. Supports opening .psd files. Contains ads (use uBlock).

Alternative to Photoshop

Inkscape

Why? It is a free and open source vector graphics editor. It offers a great set of features and supports many files formats. Supports opening Adobe Illustrator files (.ai)

Alternative to Illustrator

DaVinci Resolve

Why? Resolve (not Resolve Studio) is a free and fully fledged video editor with basically every thing you need. Requires an account. Supports all major platforms.

Alternative to Premiere
